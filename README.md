# flutter_framework

基于实战开发的FLUTTER框架

## Getting Started

```
#### version
flutter: 2.0.5
dart: 2.12.3

#### how to run
1. get pub
flutter pub get
2. model build
install plugin FlutterJsonBeanFactory
click build/FlutterBeanFactory
3. database build
flutter pub run build_runner build
4. apk build
flutter build apk
```

#### auto generate app icons
```
flutter pub run flutter_launcher_icons:main
```
